# Contributing

Thanks for contributing to `react-admin`.

## Development Setup

```bash
pnpm install
pnpm dev
```

## Branch and Commit

- Create feature branch from `main`
- Follow Conventional Commits:
  - `feat:`
  - `fix:`
  - `docs:`
  - `chore:`

## Pull Request Checklist

- Keep PRs focused and small
- Run checks locally:
  - `pnpm lint`
  - `pnpm build`
- Update docs when behavior changes
- Add screenshots for UI changes

## Code Style

- TypeScript-first changes
- Reuse existing components and hooks/stores
- Keep files and functions small and readable
